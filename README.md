# Attention-Seekers
ECE 284 Project

## Github instructions
- **Git clone:** obtain a copy of this repository **for the first time**  
`git clone https://github.com/AZA-2003/Attention-Seekers.git`
- **Git pull:** obtain an updated version with all committed changes; this is done after the **git clone**  
`git pull`
- **Git add:** add new contents or files; this is done to involve newly added files or folders  
`git add <folders or files of choice>`  
To git add all the files: `git add .`
- **Git commit:** commit any changes made before pushing; the message summarizes the commit  
`git commit -m <message>`
- **Git push:** push any and all committed changes back to the repository branch  
`git push origin master`

## Directory Structure

- **baseline**: Initial work on vanilla version.
- **hardware**: An extension of vanilla version with a controller and further verification.
- **part_2**  : Work on output stationary mapping.
- **software**: All work related to model development and strategies on quantization, pruning, etc.
